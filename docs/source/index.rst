``swxg``
==========================

``swxg`` is a Python library for modern stochastic weather generation. It synthesizes arbitrarily-long sequences of spatially- and temporally-correlated precipitation and temperature variables from a set of given observations using hidden Markov models, hydroclimatic copulae, and *k*-NN disaggregation techniques.

.. note::

   This project is under active development.
